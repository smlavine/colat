# colat

Simple program to show what hexadecimal colors actually look like.

I wrote this a long time ago, and there is much that can be improved.

# Usage

`colat <colors>`
Colat supports "#RGB" and "#RRGGBB" syntaxes.

# Building

Simply run ```make```.
